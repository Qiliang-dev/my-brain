## 🔧 MCP工具集成

```
MCP工具集成架构：

AI模型 ←→ MCP协议 ←→ 各种功能工具 ←→ 数据服务

具体工具：

1. 📊 问卷调查工具
   ├── get_survey_questions: 获取问卷问题
   ├── analyze_user_profile: 分析用户档案
   ├── generate_learning_path_from_survey: 基于问卷生成学习路径
   ├── get_survey_statistics: 获取问卷统计
   └── recommend_survey_questions: 推荐问卷问题

2. 🎯 定制化工具
   ├── get_personalized_path: 获取个性化路径
   ├── get_next_recommendation: 获取下一个推荐
   └── allocate_learning_task: 分配学习任务

3. 🧠 记忆管理工具
   ├── store_learning_memory: 存储学习记忆
   ├── retrieve_memories: 检索记忆
   └── schedule_review: 安排复习

4. 📚 任务分配工具
   ├── 任务优先级计算
   ├── 依赖关系检查
   ├── 时间调度优化
   └── 动态调整机制

数据流向：
用户输入 → MCP工具 → 数据处理 → 结果返回 → AI模型 → 用户输出
```
