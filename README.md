# My Brain - 个人知识管理系统 / Personal Knowledge Management System

## 项目概述 / Project Overview

这是一个个人知识管理系统，用于组织和学习不同主题的知识。系统采用双语标注（中文/英文），支持多项目并行学习。

This is a personal knowledge management system for organizing and learning knowledge on different topics. The system uses bilingual annotations (Chinese/English) and supports parallel learning of multiple projects.

## 学习内容示例 / Learning Content Examples

### ISTQB Foundation 学习内容 / ISTQB Foundation Learning Content

#### 七个测试原则 / Seven Testing Principles
1. **测试显示缺陷的存在，而不是它们的缺失 / Testing shows the presence of defects, not their absence**
2. **穷尽测试是不可能的 / Exhaustive testing is impossible**
3. **早期测试节省时间和金钱 / Early testing saves time and money**
4. **缺陷聚集在一起 / Defects cluster together**
5. **警惕农药悖论 (pesticide paradox) / Beware of the pesticide paradox**
6. **测试依赖于上下文 / Testing is context dependent**
7. **无错误谬论 (Absence-of-errors fallacy) / Absence-of-errors is a fallacy**

#### 测试活动 / Test Activities
1. **测试计划 (Test Planning)** / Test Planning
2. **测试监控和控制 (Test Monitoring and Control)** / Test Monitoring and Control
3. **测试分析 (Test Analysis)** / Test Analysis
4. **测试设计 (Test Design)** / Test Design
5. **测试实施 (Test Implementation)** / Test Implementation
6. **测试执行 (Test Execution)** / Test Execution
7. **测试完成 (Test Completion)** / Test Completion
